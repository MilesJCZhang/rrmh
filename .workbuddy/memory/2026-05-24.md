## 修复 /v1/payment/create 500 错误（xmlToJson 解析失败）

### 根因
`xmlToJson()` 函数没有正确处理微信支付 XML 响应中的外层 `<xml>...</xml>` 包裹标签，
导致整个响应被解析为一个 key=`"xml"` 的字段，而不是解析内部各个标签（`return_code`、`result_code` 等）。

### 修复内容
**文件**: `/home/ubuntu/renrenmeihao-api/src/routes/payment.ts`

在 `xmlToJson()` 函数中加入外层 `<xml>` 剥离逻辑：
```typescript
const xmlToJson = (xml: string): Record<string, string> => {
  const result: Record<string, string> = {};
  // 先剥离 XML 声明和外层 <xml> 包裹
  let cleaned = xml.replace(/<\?xml.*?\?>/s, '').trim();
  const outer = cleaned.match(/^<xml>([\s\S]*?)<\/xml>$/);
  if (outer) cleaned = outer[1];
  
  const tagRe = /<(\w+)>([\s\S]*?)<\/\1>/g;
  let m: RegExpExecArray | null;
  while ((m = tagRe.exec(cleaned)) !== null) {
    let val = m[2].trim();
    const cdata = val.match(/^<!\[CDATA\[(.*?)\]\]>$/s);
    if (cdata) val = cdata[1];
    result[m[1]] = val;
    if (m[0].length === 0) break;
  }
  return result;
};
```
