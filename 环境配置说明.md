## 使用前必填项

### 微信小程序配置
在 `.env` 文件中填写真实的微信小程序配置：
```bash
WX_APPID=你的微信小程序AppID
WX_SECRET=你的微信小程序AppSecret
```

### 微信支付配置（如需支付功能）
在 `.env` 文件中填写微信支付商户信息：
```bash
WECHAT_MCH_ID=你的商户号
WECHAT_API_KEY=你的API密钥
WECHAT_APPID=你的微信AppID
```

### JWT_SECRET 安全建议
修改 `.env` 中的 `JWT_SECRET` 为一个复杂的随机字符串：
```bash
JWT_SECRET=your_complex_random_string_here
```

## 常见问题

### Q: 小程序端无法请求后端 API？
**A**: 检查是否在微信公众平台配置了服务器域名，且域名已备案并支持 HTTPS。

### Q: 微信登录失败？
**A**: 检查 `.env` 中的 `WX_APPID` 和 `WX_SECRET` 是否正确。

### Q: 支付功能无法使用？
**A**: 检查 `.env` 中的微信支付相关配置是否正确，并确保微信支付商户号已开通。
