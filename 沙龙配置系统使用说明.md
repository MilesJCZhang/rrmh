# 沙龙配置系统使用说明

## 概述

沙龙配置系统用于在小程序端区分和配置三种沙龙类型：
1. **常规沙龙**（`mixed`） - 3男3女分组模式
2. **男推荐官沙龙**（`male_salon`） - 仅男性推荐官参加
3. **女推荐官沙龙**（`female_salon`） - 仅女性推荐官参加

通过配置系统，可以灵活配置不同类型的沙龙的UI主题、功能权限、报名流程等，而无需修改代码。

## 配置系统核心文件：`salon-config.js`

### 1. 沙龙类型常量

```javascript
const SALON_TYPES = {
  MIXED: 'mixed',           // 常规沙龙
  MALE_SALON: 'male_salon',     // 男推荐官沙龙
  FEMALE_SALON: 'female_salon', // 女推荐官沙龙
};
```

### 2. 配置结构

每个沙龙类型的配置包含以下部分：

#### 2.1 基本信息

```javascript
{
  type: 'male_salon',
  name: '男推荐官主体沙龙',
  shortName: '男沙龙',
  description: '男士推荐官主办・专属圈层交流',
}
```

#### 2.2 主题配置（`theme`）

用于配置沙龙列表页和详情页的UI主题：

```javascript
theme: {
  color: '#1565C0',           // 主题色
  lightColor: '#E3F2FD',      // 浅色背景
  gradient: 'linear-gradient(135deg, #1565C0, #42A5F5)', // 渐变色
  bannerBg: 'linear-gradient(135deg, #1565C0, #42A5F5)', // Banner背景
  icon: '🏋‍♂️',            // 图标
  emoji: '💙',                // Emoji
}
```

#### 2.3 页面配置（`page`）

用于配置沙龙相关页面的路径：

```javascript
page: {
  list: '/subpackages/activity/pages/salon-list/salon-list?type=male_salon',
  detail: '/subpackages/activity/pages/salon-detail/salon-detail?type=male_salon',
  create: '/subpackages/activity/pages/salon-create/salon-create?type=male_salon',
  navTitle: '男推荐官主体沙龙',
}
```

#### 2.4 功能配置（`features`）

用于配置沙龙的功能特性：

```javascript
features: {
  isGrouped: false,             // 是否分组
  maxRecommenders: 9,          // 推荐官上限
  maxCompanionsPerPerson: 2,    // 每人最多随行人数
  totalCap: 27,                // 单场总人数封顶
  allowCompanions: true,        // 是否允许随行人员
  requireScoreTier: false,      // 是否需要评分等级
  allowedTiers: ['all'],       // 允许的评分等级
  weeklyLimit: 1,              // 每周报名次数限制
  needApproval: true,           // 是否需要审核
  showScoreFilter: false,       // 是否显示评分筛选
  weekDay: [1, 2, 3, 4, 5, 6, 7], // 每周几可以举办
}
```

#### 2.5 报名流程配置（`registration`）

用于配置报名流程：

```javascript
registration: {
  needPersonalInfo: true,       // 是否需要个人资料
  needCompanionInfo: true,      // 随行人员是否需要详细资料
  needPayment: true,            // 是否需要支付
  defaultFee: 399,             // 默认报名费
  allowFree: false,             // 是否允许免费
  companionFields: [            // 随行人员需要填写的字段
    'name',
    'phone',
    'gender',
    'age',
    'industry',
  ],
}
```

#### 2.6 权限配置（`permissions`）

用于配置谁可以创建和参加沙龙：

```javascript
permissions: {
  whoCanCreate: ['male_matchmaker'], // 谁可以创建
  whoCanJoin: 'specific_gender',    // 谁可以参加
  genderRestriction: 'male',        // 性别限制
}
```

## 使用方式

### 1. 在页面中引入配置系统

```javascript
const { SALON_TYPES, getSalonConfig } = require('../../utils/salon-config');
```

### 2. 获取沙龙配置

```javascript
// 根据沙龙类型获取配置
const config = getSalonConfig('male_salon');

// 使用配置
const themeColor = config.theme.color;
const maxRecommenders = config.features.maxRecommenders;
const defaultFee = config.registration.defaultFee;
```

### 3. 获取所有沙龙类型配置

```javascript
const { getAllSalonConfigs } = require('../../utils/salon-config');
const allConfigs = getAllSalonConfigs();
```

### 4. 获取沙龙类型列表（用于首页入口配置）

```javascript
const { getSalonTypeList } = require('../../utils/salon-config');
const salonTypeList = getSalonTypeList();

// 返回结果：
// [
//   {
//     key: 'mixed',
//     type: 'mixed',
//     name: '圈层主题沙龙',
//     shortName: '主题沙龙',
//     description: '线下志趣交流・品质圈层聚会',
//     icon: '🎉',
//     color: '#C8102E',
//     page: '/subpackages/activity/pages/salon-list/salon-list',
//     gridKey: 'salon',
//   },
//   // ... 其他类型
// ]
```

### 5. 根据首页gridKey获取沙龙配置

```javascript
const { getSalonConfigByGridKey } = require('../../utils/salon-config');
const config = getSalonConfigByGridKey('maleSalon'); // 对应男推荐官沙龙
```

### 6. 检查用户权限

```javascript
const { checkJoinPermission, checkCreatePermission } = require('../../utils/salon-config');

// 检查用户是否可以参加沙龙
const joinResult = checkJoinPermission('male_salon', userInfo);
if (!joinResult.allowed) {
  wx.showToast({ title: joinResult.reason, icon: 'none' });
}

// 检查用户是否可以创建沙龙
const createResult = checkCreatePermission('male_salon', userInfo);
if (!createResult.allowed) {
  wx.showToast({ title: createResult.reason, icon: 'none' });
}
```

## 沙龙列表页配置

沙龙列表页（`salon-list.js`）使用配置系统来：
1. 根据沙龙类型设置主题色
2. 根据配置读取功能配置（如 `totalCap`、`maxRecommenders` 等）
3. 控制UI显示（如是否显示评分筛选）

### 关键代码

```javascript
// 引入配置系统
const { getSalonConfig, SALON_TYPES } = require('../../../../utils/salon-config');

// 在 onLoad 中读取配置
onLoad(options) {
  const salonType = options.type || '';
  
  // 使用配置系统获取主题配置
  const config = getSalonConfig(salonType);
  const theme = config ? config.theme : null;
  
  const data = {
    salonType,
    // 从配置中读取功能配置
    totalCap: config ? config.features.totalCap : 27,
    maxRecommenders: config ? config.features.maxRecommenders : 9,
    showScoreFilter: config ? config.features.showScoreFilter : true,
  };
  
  // 设置主题色
  if (theme) {
    data.themeColor = theme.color;
    data.themeLightColor = theme.lightColor;
    data.themeGradient = theme.gradient;
    data.themeBannerBg = theme.bannerBg;
    data.navTitle = config.name || '线下沙龙';
    wx.setNavigationBarTitle({ title: data.navTitle });
  }
  
  this.setData(data);
  this.loadSalons();
}
```

## 如何添加新沙龙类型

如果需要添加新的沙龙类型（如"情侣沙龙"），只需按照以下步骤操作：

### 1. 在 `salon-config.js` 中添加新类型常量

```javascript
const SALON_TYPES = {
  MIXED: 'mixed',
  MALE_SALON: 'male_salon',
  FEMALE_SALON: 'female_salon',
  COUPLE_SALON: 'couple_salon', // 新增：情侣沙龙
};
```

### 2. 在 `SALON_TYPE_CONFIG` 中添加配置

```javascript
[SALON_TYPES.COUPLE_SALON]: {
  type: SALON_TYPES.COUPLE_SALON,
  name: '情侣沙龙',
  shortName: '情侣沙龙',
  description: '情侣专属活动・甜蜜互动交流',
  
  theme: {
    color: '#FF6B6B',
    lightColor: '#FFE0E0',
    gradient: 'linear-gradient(135deg, #FF6B6B, #FF8E8E)',
    bannerBg: 'linear-gradient(135deg, #FF6B6B, #FF8E8E)',
    icon: '💑',
    emoji: '❤️',
  },
  
  // ... 其他配置
}
```

### 3. 在首页添加入口

在 `index.wxml` 中添加新的入口卡片，并在 `index.js` 中配置跳转逻辑。

### 4. 更新后端API

确保后端API支持新的沙龙类型。

## 常见问题

### 1. 如何修改沙龙主题色？

修改 `salon-config.js` 中对应沙龙类型的 `theme.color` 字段。

### 2. 如何修改沙龙报名费？

修改 `salon-config.js` 中对应沙龙类型的 `registration.defaultFee` 字段。

### 3. 如何限制用户每周报名次数？

修改 `salon-config.js` 中对应沙龙类型的 `features.weeklyLimit` 字段。设置为 `0` 表示无限制。

### 4. 如何控制是否显示评分筛选？

修改 `salon-config.js` 中对应沙龙类型的 `features.showScoreFilter` 字段。设置为 `true` 显示，设置为 `false` 隐藏。

---

## 总结

通过使用沙龙配置系统，可以：
1. **集中管理**所有沙龙类型的配置
2. **快速切换**不同沙龙类型的UI主题
3. **灵活控制**功能权限和报名流程
4. **易于扩展**新的沙龙类型

无需修改多个文件，只需更新 `salon-config.js` 中的配置即可。
