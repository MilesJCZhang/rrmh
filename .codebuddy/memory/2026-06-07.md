### 报名 API 404 修复
- **问题**：生产服务器 `/v1/salon/:id/signup` 路由使用 `req.app.handle()` 转发到 `/join`，但转发机制断裂导致 404
- **修复**：`services/api.js` 中 `API.SALON.JOIN` 路径从 `/v1/salon/:id/signup` 改为 `/v1/salon/:id/join`
- **说明**：生产服务器 `/join` 为简化版（Prisma模型，仅存activityId+userId+status），不走 `/signup` → `/join` 转发链
