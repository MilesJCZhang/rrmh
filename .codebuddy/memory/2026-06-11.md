## 管理后台退出后无法登录修复（2026-06-11 17:21）
- **问题根源**：token 存储双源不同步
  - `auth.util.ts` 使用 `localStorage.getItem('token')` 检查登录状态
  - 但 `Login.tsx` 使用 Zustand store（`useAuthStore`）将 token 持久化到 `auth-storage` 键
  - `PrivateRoute` 和 `axios` 拦截器从 `localStorage.getItem('token')` 读取，始终为 null
  - 结果：登录成功（Zustand 存到 `auth-storage`）→ `PrivateRoute` 检查失败 → 重定向回登录页
- **修复文件**：`/home/ubuntu/renrenmeihao-api/admin-src/src/utils/auth.util.ts`
  - `getToken()` → `useAuthStore.getState().token`（从 Zustand 读取）
  - `setToken()` → `useAuthStore.getState().setToken(token)`
  - `removeToken()` → `useAuthStore.getState().logout()`
  - `isAuthenticated()` → `!!useAuthStore.getState().token`
  - `logout()` → `useAuthStore.getState().logout()` + 跳转
- **同步修复**：`axios.config.ts`
  - 请求拦截器：`Ou.getState().token` 替代 `localStorage.getItem('token')`
  - 响应拦截器(401)：`useAuthStore.getState().logout()` 替代 `localStorage.removeItem('token')`
- **构建部署**：`npm run build` → `sudo cp -r dist/* /var/www/admin/`
- **验证**：构建产物中 `isAuthenticated` 变为 `() => !!Ou.getState().token`
