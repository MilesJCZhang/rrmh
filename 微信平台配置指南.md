### 步骤2：配置服务器域名

**路径**：左侧菜单 → 「开发」→「开发管理」→「开发设置」→「服务器域名」

点击「修改」按钮，添加以下域名：

#### ✅ request 合法域名（必填）
```
https://rrmhdate.cn
```
**用途**：小程序发起的网络请求（wx.request）

**需要添加的接口**：
- `https://rrmhdate.cn/v1/auth/wechat-login` - 微信登录
- `https://rrmhdate.cn/v1/users/profile` - 用户信息
- `https://rrmhdate.cn/v1/payment/create` - 支付下单
- `https://rrmhdate.cn/v1/matchmaker/*` - 媒婆相关接口

#### ✅ socket 合法域名（如需聊天功能）
```
wss://rrmhdate.cn
```
**用途**：WebSocket连接（wx.connectSocket）

#### ✅ uploadFile 合法域名（如需上传图片）
```
https://rrmhdate.cn
```
**用途**：上传文件（wx.uploadFile）

#### ✅ downloadFile 合法域名（如需下载文件）
```
https://rrmhdate.cn
```
**用途**：下载文件（wx.downloadFile）

**配置示例截图描述**：
```

├─────────────────────────────────────────┤

```

⚠️ **注意**：
- 域名必须使用 HTTPS（wss）
- 域名不能包含端口号
- 域名必须已备案
- 每个类别最多可添加20个域名

### 步骤4：配置消息推送（可选，用于支付通知）

**路径**：「开发」→「开发管理」→「开发设置」→「消息推送」

如果需要在后端接收微信的异步通知（如支付结果），需要配置：

| 配置项 | 值 |
|--------|-----|
| URL | `https://rrmhdate.cn/v1/payment/notify` |
| Token | 自定义令牌（如：`renrenmeihao_token_2024`） |
| EncodingAESKey | 点击「随机生成」 |
| 消息加密方式 | 安全模式（推荐） |

**后端对应代码**（在 `routes_payment_updated.js` 中）：
```javascript
// 微信支付通知接口
router.post('/notify', async (req, res) => {
  try {
    const result = await wxpay.notify(req.body);
    
    // 处理支付结果
    if (result.return_code === 'SUCCESS') {
      await db.updateOrder(result.out_trade_no, {
        status: 'paid',
        transaction_id: result.transaction_id,
      });
    }
    
    res.send(`
      <xml>
        <return_code><![CDATA[SUCCESS]]></return_code>
        <return_msg><![CDATA[OK]]></return_msg>
      </xml>
    `);
  } catch (error) {
    console.error('支付通知处理失败:', error);
    res.send(`
      <xml>
        <return_code><![CDATA[FAIL]]></return_code>
        <return_msg><![CDATA[${error.message}]]></return_msg>
      </xml>
    `);
  }
});
```

## 🔐 微信支付配置

如果需要使用微信支付功能，还需要以下配置：

### 1. 开通微信支付

**路径**：「微信支付」→「开通微信支付」

- 选择「新申请」或「绑定已有商户号」
- 按照指引提交资料
- 等待审核（通常1-3个工作日）

### 2. 配置支付目录（H5支付用）

**路径**：微信支付商户平台 → 产品中心 → 开发配置

添加支付授权目录：
```
https://rrmhdate.cn/pages/payment/
```

### 3. 设置API密钥

**路径**：微信支付商户平台 → API安全 → API密钥

- 点击「设置密钥」
- 输入32位密钥（可随机生成）
- **重要**：将密钥填写到 `.env` 文件的 `WECHAT_API_KEY`

## 🐛 常见问题

### Q1: 域名配置后小程序仍无法请求？

**可能原因**：
1. 域名未备案 ❌
2. SSL证书无效或过期 ❌
3. 域名解析未生效（等待DNS传播，最多48小时）⏳
4. 小程序代码中的域名与配置不一致 ❌

**解决方法**：
```bash
# 检查域名是否备案
whois rrmhdate.cn

# 检查SSL证书
openssl s_client -connect rrmhdate.cn:443 -servername rrmhdate.cn

# 测试接口
curl -v https://rrmhdate.cn/health
```

### Q2: 提示"不在以下 request 合法域名列表中"？

**原因**：小程序代码中请求的域名未在公众平台配置

**解决方法**：
1. 检查代码中的请求域名
2. 在公众平台「服务器域名」中添加该域名
3. 重启小程序开发工具

### Q3: 微信登录失败，提示"code been used"？

**原因**：微信登录code只能使用一次

**解决方法**：
- 每次登录都需要重新调用 `wx.login()` 获取新code
- 不要在代码中缓存code

### Q4: 支付报错"商户号不存在"？

**原因**：`.env` 中的商户号配置错误

**解决方法**：
1. 登录微信支付商户平台
2. 复制正确的商户号（mch_id）
3. 更新 `.env` 文件并重启服务
